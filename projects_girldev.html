My Projects

The majority of my past programming projects have been related to doing astronomy, but I've been working hard lately to start applying my programming skills to other realms - like website development and playing with public data on the internet.

Science Research Projects

  • Studying Galaxy Clusters with Gravitational Lensing: This ongoing project is a statistical analysis of around 20,000 clusters of galaxies, and over a million individual galaxies, using a technique called weak gravitational lensing. Lensing is a purely statistical way of measuring masses using the way that light from distant galaxies is deflected. I've written thousands of lines of python code (a language I've transitioned to over the last 1.5 years), which I integrate with C programs when I need speed. I employ techniques such as bootstrapping (for error bars and covariance matrices), randomization (to check for systematics), and parallel processing (with openMP).
  • Magnified Galaxies: This was an analysis of the dark matter halos of 44 galaxy clusters using > 30,000 background galaxies that were being gravitationally magnified, a signal that can only be teased out of the data through statistical averaging over a huge ensemble of data. I programmed thousands of lines of IDL and C to do the measurement, fit and compare different theoretical models, and estimate uncertainties using the jackknife method (similar to bootstrapping).
  • Image Simulations at NASA JPL: This 2007 summer project was when I first learned basic programming (outside of a university class) as well as how to use the command line on a linux machine. I coded some small IDL programs to simulate the relative image quality that could be achieved for space-based telescopes of different sizes (ie, prices) to help plan the requirements for a future mission.
